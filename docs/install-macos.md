# macOS Installation Guide

## Homebrew

```bash
brew tap keathmilligan/tap
brew install keathmilligan/tap/gfm-hotview
```

Stay up-to-date with `brew upgrade gfm-hotview`.

## Shell Installer

```bash
curl -fsSL https://packages.keathmilligan.net/gfm-hotview/install.sh | sh
```

This will install `gfm-hotview` into `~/.local/bin`.

## dmg Installer

Download the signed `.dmg` installer for your platform architecture (Apple
Silicon `aarch64` or Intel `x86_64`) directly from the [GitHub
Releases](https://github.com/keathmilligan/gfm-hotview/releases) page.

## Binary

Download the macOS binary archive for your architecture (Apple Silicon
`aarch64` or Intel `x86_64`) from the [GitHub
Releases](https://github.com/keathmilligan/gfm-hotview/releases) page.

Extract the `gfm-hotview` binary into a directory in your `PATH`.
